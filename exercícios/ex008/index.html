Treho de Código

Para escrever um código de uma forma que seja de facil leitura e que indentifique-o, usa-se a tag CODE.
Exemplos:
// Solicitar ao usuário para inserir o primeiro número let numero1 = prompt("Digite o primeiro número:"); // Converter a entrada do usuário para um número numero1 = parseFloat(numero1); // Solicitar ao usuário para inserir o segundo número let numero2 = prompt("Digite o segundo número:"); // Converter a entrada do usuário para um número numero2 = parseFloat(numero2); // Calcular a soma dos dois números let soma = numero1 + numero2; // Exibir o resultado no console console.log("A soma de " + numero1 + " e " + numero2 + " é: " + soma);

Para o código ficar igual como foi codado usa-se a tag PRE, juntamente com o CODE, para assim, o texto fica pré-formatado como foi escrito.
Exemplo: 

// Solicitar ao usuário para inserir o primeiro número
        let numero1 = prompt("Digite o primeiro número:");
        
        // Converter a entrada do usuário para um número
        numero1 = parseFloat(numero1);
        
        // Solicitar ao usuário para inserir o segundo número
        let numero2 = prompt("Digite o segundo número:");
        
        // Converter a entrada do usuário para um número
        numero2 = parseFloat(numero2);
        
        // Calcular a soma dos dois números
        let soma = numero1 + numero2;
        
        // Exibir o resultado no console
        console.log("A soma de " + numero1 + " e " + numero2 + " é: " + soma);

Abreviações

Para colocar abreviações que vá além de só as siglas, usa-se a tag ABBR. Nela é possível colocar o name do que está sendo abreviado, assim, quando o cursor do mause passar em cima da sigla aparecerá o seu nome não abreviado.
Exemplo:
A ONU, proíbe explosões nucleares em qualquer lugar da Terra, seja em sua superfície, atmosfera, debaixo d'água ou subsolo.

Texto Invertido

Para nverter um texto, usa-se o tag BDO, nela em DIR, escolhe a direção:
rtl= direita para esquerda - o que na realidade não faz nada
ltr= esquerda par direita.
Exemplos:
texto invertido ltr
texto invertido rtl
